# xunzip2 - A minimal unzip library for the original Xbox

xunzip2 is a simple unzip library for the original [Microsoft Xbox](https://en.wikipedia.org/wiki/Xbox_(console)) leveraging [unzipLIB](https://github.com/bitbank2/unzipLIB).



# Authors

* CrunchBite ([BlueSky](https://bsky.app/profile/crunchbite.teamxlink.co.uk))
* [bitbank2](https://github.com/bitbank2/unzipLIB) (for unzipLIB)
